# UNP_PROJECT_BI_WITH_POWER_BI_FINANCIAL_DATASET_ANALYSIS

I did this project in collaboration with UNP, where we analyzed a company financial dataset using Power BI. The aim was to study key performance indicators like revenue, profit, units sold, discounts, and profit margin. We created interactive dashboards to analyze performance across different months, countries, products, and segments. This helped us identify trends and understand how factors like discounts impact profitability. From our analysis, we found issues such as losses in the enterprise segment, high manufacturing costs in some products, and low revenue during certain months. Based on these insights, we suggested solutions like reducing discounts, improving pricing strategies, conducting internal audits, and using promotions during low-sales periods. Overall, the project helped us understand how data visualization can support better business decisions.
